## Technologies Used
- React
- Firebase
- TailwindCSS
## Installation
Install NodeJS in your computer
If you've cloned this repository, open in terminal and typing in your terminal with command :
```
npm install
npm install craco
```

## Running
For Running :
```
npm start
```
